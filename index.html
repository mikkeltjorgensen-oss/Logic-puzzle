Kaptajnens hovedvrider


Ledetråde

1. Søren arbejder altid senere end Salvie

2. Søren eller Svend fanger torsk, den anden fanger makrel

3. Torsk bliver aldrig fanget på den tidligste arbejdstid

4. Fisken som bliver fanget mandag er enten skrubbe eller torsk

5. Salvie eller Svend arbejder enten fra 05-10 eller 10-15

6. Sømændende som arbejder fra 05-10 og 15-20 arbejder enten tirsdag eller onsdag

7. Båden sejler kun 15-20 om tirsdagen

8. Enten fanger Søren torsk Tirsdag eller Onsdag

Sådan spiller du

1. Kun en sømand kan arbejde en specifik dag, en arbejdstid og fange en type fisk

2. Der er ingen gengangere

3. Sæt X hvor det ikke kan lade sig gøre

4. Sæt O hvor det kan lade sig gøre
